# 🎭 Random Excuse Extension

A lightweight Chrome extension that generates funny, slightly questionable excuses with one click.

---

## 🚀 What it does
Click the extension button and instantly get a random excuse for:
- school  
- work  
- life situations you don’t want to explain  

---

## 🧩 How to install (manual)
1. Download or clone this repository  
2. Go to `chrome://extensions/` in Chrome  
3. Enable **Developer mode** (top right)  
4. Click **Load unpacked**  
5. Select the project folder  

---

## ✨ Features
- Random excuse generator  
- Clean popup interface  
- Fast and lightweight  
- Slightly chaotic energy
